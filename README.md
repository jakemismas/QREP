# QREP

QREP (Quilt Reverse Engineering Platform) is an open-source Python library and
CLI that reverse engineers quilts from photographs into production-ready
patterns: cut lists, yardage, strip-piecing plans, SVG diagrams, and a PDF
pattern booklet.

v1 is under active construction. The design doc is
[qrep-design-doc.md](qrep-design-doc.md); the build contract is
[qrep-claude-code-prompt.md](qrep-claude-code-prompt.md). Work is tracked in
[GitHub Issues](https://github.com/jakemismas/QREP/issues).

This README is replaced with a full install and CLI walkthrough at the end of
Sprint 1 (issue tracked in the sprint).
